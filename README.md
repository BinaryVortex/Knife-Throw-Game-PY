# 🔪🎯 Knife Throw Game (Python)

## 📌 Overview  
This repository contains a **Knife Throw Game built using Python programming language**.  
The game is a simple arcade-style project where players throw knives at targets, focusing on timing, accuracy, and reaction speed.

---

## 🎯 Purpose

- Improve logic building and event handling  
- Learn basic physics and collision detection  
- Develop interactive arcade-style gameplay  
- Enhance programming and problem-solving skills
